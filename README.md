# sistema-turnos-electrodos
Sistema integral de gestión de turnos con kiosco táctil, panel de trabajador en tiempo real, impresión térmica automática Zebra y chatbot interactivo de inteligencia artificial desarrollado con la herramienta de automatización n8n. El acceso al chatbot es mediante una página web. Desarrollado en PHP 8 + MariaDB con arquitectura de roles.
